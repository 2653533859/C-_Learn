# 04 - 数据结构

数据结构是高效算法的基石。本模块以逐步递进的方式介绍常见线性与树形结构，配合图示、源码与练习，帮助你建立“抽象模型 ↔ 内存布局 ↔ 实战代码”的完整思维链路。

## 学习顺序
1. 数组与字符串：理解顺序存储的特点与缓存友好性
2. 结构体与联合体：设计贴近问题域的数据模型
3. 单/双链表：体会指针串联带来的灵活性
4. 栈与队列：掌握受限操作的线性结构及典型应用
5. 二叉树基础：从根节点出发构建递归与分治思维

## 资源分类
